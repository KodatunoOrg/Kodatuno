======================
Inputファイル(***.inp)
======================

NURBS曲面に関するInputファイルを作成する際は、ファイルの先頭行にu,v各方向の点の数を入力してください。
また、各点の座標の順序はuv方向の順序に則ります。以下の例を参考にしてください。

例: 六角形を生成するファイルを作成したい場合

2,3											←u,v各方向の点の数(u,v)
0,0,0										←u方向1番目、v方向1番目の点の座標(x,y,z)
50,-28.87,0							←u方向1番目、v方向2番目の点の座標(x,y,z)
100,0,0									←u方向1番目、v方向3番目の点の座標(x,y,z)
0,57.74,0								←u方向2番目、v方向1番目の点の座標(x,y,z)
50,86.6,0								←u方向2番目、v方向2番目の点の座標(x,y,z)
100,57.74,0							←u方向2番目、v方向3番目の点の座標(x,y,z)
